OpenSource Node.JS PaaS


Deploy your Node.JS applications to Nodester.com for FREE or deploy your apps to your own private cloud instance of Nodester running in your own datacenter or on Amazon EC2, Rackspace, or GoGrid!


Nodester is an open source Node.JS Platform-as-a-Service written in Node.JS with a RESTful API designed to run in any cloud without a VPS! It is the first of its kind and 100% open sourced under the GNU Affero license on GitHub.


We now support Node.JS versions 0.4.12, 0.6.12, 0.6.17, and 0.8.1!
